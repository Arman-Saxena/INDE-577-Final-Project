# **topsongs.csv — Dataset Overview**

## **Description**
The **topsongs.csv** dataset contains numerical audio features extracted from Spotify tracks. These features describe musical characteristics such as rhythm, energy, acoustic profile, and overall mood. The dataset is used across both supervised and unsupervised learning modules to analyze structure in music data and build predictive models.

---

## **Features**
All columns represent **numeric**, standardized Spotify audio attributes:

| Feature | Meaning |
|--------|---------|
| **danceability** | How suitable a track is for dancing (0–1). |
| **energy** | Intensity and activity of the track (0–1). |
| **loudness** | Overall loudness in decibels (dB). |
| **speechiness** | Presence of spoken words (0–1). |
| **acousticness** | Probability the track is acoustic (0–1). |
| **instrumentalness** | Likelihood the track has no vocals (0–1). |
| **liveness** | Detects audience presence (0–1). |
| **valence** | Musical positivity (0–1). |
| **tempo** | Estimated beats per minute (BPM). |
